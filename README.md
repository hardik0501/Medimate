# 🩺 MediMate – Your AI Doctor Assistant 👨‍⚕️💬

MediMate is an intelligent medical assistant web app built with **Streamlit** and **Gemini AI**, helping users get preliminary health advice, possible diagnoses, and nearby hospital suggestions—all with a pinch of humor. 🤖💊

---

## 🌟 Features

- 🔐 **User Login & Registration**
- 📝 **Symptom-based Diagnosis** using WebMD scraping + Gemini AI
- 🧠 **AI Medical Summary** in Hinglish (fun + informative)
- 🖼️ **Image-based Diagnosis** (skin rash, eye infection, etc.)
- 🏥 **Nearby Hospital Finder** using Google Places API
- 🔊 **Text-to-Speech** output in Hinglish (gTTS)
- 😜 **Witty Medical Jokes** to lighten the mood
- 📱 Mobile-friendly & easy-to-use UI
